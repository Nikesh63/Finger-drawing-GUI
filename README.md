🎨 Finger Drawing App (OpenCV + Python)

A real-time virtual drawing application that lets you draw
in the air using your finger and a webcam.
Built with OpenCV and NumPy, it uses simple HSV-based 
skin detection to track hand movement and create a live 
drawing experience.

✨ Features:
Draw by moving your hand in the upper half of the screen.
Change colors with a bottom-screen gesture.
Clear canvas with a left-side gesture.
Toggle background blur with a right-side gesture.
Save drawings as images.
Real-time gesture debouncing for smooth interaction.
A lightweight computer vision project demonstrating 
hand tracking, gesture control, and interactive UI using
only OpenCV.
